# 🍕 Pizza Hut Sales Analysis | Power BI + SQL Project

## 📌 Project Overview
This project is an end-to-end data analytics solution built using SQL and Power BI. The goal is to analyze Pizza Hut sales data and generate meaningful business insights through interactive dashboards.

The project covers the complete data analysis workflow — from raw data processing in SQL to visualization and storytelling in Power BI.

---

## 🎯 Objectives
- Analyze sales performance across different categories and sizes
- Identify top-performing and underperforming products
- Track daily and monthly sales trends
- Build key performance indicators (KPIs) for business decision-making

---

## 🛠️ Tools & Technologies Used
- **SQL** – Data extraction, transformation, and KPI creation  
- **Power BI** – Data visualization and dashboard development  
- **Excel** – Initial data handling and preprocessing  

---

## 📂 Project Workflow

### 1. Data Understanding
- Explored raw dataset containing pizza sales transactions
- Identified key columns such as:
  - Order date
  - Category
  - Size
  - Revenue
  - Quantity

### 2. Data Processing in SQL
- Imported dataset into SQL database
- Wrote queries to generate:
  - KPIs (Total Revenue, Total Orders, Average Order Value)
  - Daily & Monthly sales trends
  - % Sales by Category & Size
  - Top 5 & Bottom 5 products

### 3. Data Visualization in Power BI
- Connected Power BI to SQL database
- Performed data cleaning using Power Query
- Created interactive dashboards including:
  - KPI Cards
  - Sales Trends (Daily & Monthly)
  - Category & Size Analysis
  - Best & Worst Sellers

---

## 📊 Key Insights

- 🍕 Classic pizzas contributed the highest sales
- 📦 Large-sized pizzas generated the most revenue
- 📅 Sales peaked during weekends (especially Friday & Saturday evenings)
- 📈 Highest order volumes observed in January and July
- 🥇 Identified top-performing and least-performing pizza items
